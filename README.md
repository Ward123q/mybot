# 🔒 Политика конфиденциальности CHAT GUARD

**Последнее обновление:** Март 2026

---

## 📋 Общие положения

CHAT GUARD — Telegram бот для модерации чатов. Настоящая политика конфиденциальности описывает какие данные собираются, как используются и как защищаются.

Используя бота вы соглашаетесь с данной политикой.

---

## 📊 Какие данные собираются

### Автоматически при использовании бота:
| Данные | Зачем |
|--------|-------|
| Telegram ID | Идентификация пользователя |
| Имя пользователя | Отображение в командах |
| ID чата | Привязка данных к чату |
| Счётчик сообщений | Система XP и уровней |
| Дата последней активности | Функция стрика |

### По действиям модераторов:
| Данные | Зачем |
|--------|-------|
| История варнов | Система предупреждений |
| История банов | Учёт нарушений |
| Заметки модераторов | Модерация чата |

### По желанию пользователя:
| Данные | Зачем |
|--------|-------|
| Био профиля | Команда /setbio |
| Настроение | Команда /setmood |
| День рождения | Поздравления |

---

## ❌ Что НЕ собирается

- ✅ Содержимое личных сообщений
- ✅ Медиафайлы (фото, видео, документы)
- ✅ Голосовые сообщения
- ✅ Пароли или платёжные данные
- ✅ Геолокация
- ✅ Контакты из телефонной книги

---

## 🔐 Как данные защищаются

- Данные хранятся в зашифрованной SQLite базе
- Автоматический бэкап каждые 6 часов
- Доступ к данным только у владельца бота
- Данные не передаются третьим лицам
- Данные не продаются и не используются в рекламе

---

## 👥 Кто имеет доступ к данным

| Роль | Доступ |
|------|--------|
| Пользователь | Только свои данные |
| Модератор | Статистика и история нарушений |
| Администратор | Данные чата |
| Владелец бота | Полный доступ для тех. поддержки |

---

## 🗑 Удаление данных

Вы можете запросить удаление своих данных:
1. Напишите боту в ЛС `/deleteme`
2. Владелец обработает запрос в течение 48 часов

При удалении удаляются:
- Профиль и настройки
- История XP и уровней
- Список друзей и отношений
- Анонимные сообщения

**Не удаляются:**
- История нарушений (варны/баны) — хранится для безопасности чата
- Записи в журнале модерации

---

## 🍪 Cookies и трекеры

Бот **не использует** cookies, трекеры или пиксели слежения.

Веб-дашборд использует только сессионные cookies для авторизации администраторов.

---

## 🔄 Изменения политики

О значимых изменениях политики будет сообщено через бота.

Актуальная версия всегда доступна по команде `/privacy`.
